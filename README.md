## Reynaldo Quispe — Portfolio

Personal portfolio built with **Next.js**, focused on performance, smooth scrolling/animations, and a clean UX.

### Highlights

- **Sections**: Whoami, Experience, Projects, Stack
- **AI Match**: an interactive “smart match” panel to compare role requirements with my profile
- **Motion**: GSAP + ScrollTrigger/ScrollSmoother
- **Images**: WebP assets for smaller bundle size

### Tech stack

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Animation**: GSAP (ScrollTrigger / ScrollSmoother)
- **State & data**: Zustand (plus project-specific hooks)

### Local development

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

### SEO / Social sharing

Set these env vars for correct previews:

- `NEXT_PUBLIC_SITE_URL` (e.g. `https://your-domain.com`)
- `NEXT_PUBLIC_TWITTER_HANDLE` (optional)

See `.env.example`.
